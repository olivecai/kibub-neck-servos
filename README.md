# kibub-neck-servos
A small and simple repository to interface with the Feetech servos for the Kibub robot. This repository contains scripts to initialize the robot neck servos, calibrate them, and for the OpenClaw agent to easily move the head around for facial recognition, searching for objects, etc.

To use the Feetech servo library as a module, run this in your terminal:

export PYTHONPATH=$PWD/python-st3215:$PYTHONPATH
